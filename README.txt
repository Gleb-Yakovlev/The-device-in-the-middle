Реализация простого приложения "посередине". Отправитель посылает команду в сеть Ethernet на коммутационное устройство, ее перехватывает приложение "посередине", подменяет команду и отправляет на коммутатор. 
Коммутатор ловит команду и посылает ответ Отправителю, но ответ перехватывается приложением "посередине", подменяется на нужный и отправляется Отправителю. 
Eng(auto)
The implementation of a simple application "in the middle". The sender sends a command to the Ethernet network to the switching device, it is intercepted by the application "in the middle", 
replaces the command and sends it to the switch. 
The switch catches the command and sends a response to the Sender, but the response is intercepted by the application "in the middle", replaced with the desired one and sent to the Sender.
